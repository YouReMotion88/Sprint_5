# Sprint_5

Работа выполнена в рамках финально проекта 5 спринта "UI-тестирование" Яндекс Практикума.

## Описание проекта

Автоматизированные UI-тесты для веб-приложения Stellar Burgers.

В проекте реализованы проверки:

- регистрация пользователя;
- авторизация различными способами;
- переход в личный кабинет;
- переход из личного кабинета в конструктор;
- переход по логотипу Stellar Burgers;
- выход из аккаунта;
- переход между разделами конструктора.

## Используемые технологии

- Python
- Pytest
- Selenium WebDriver


## Запуск тестов

python -m pytest -v
