Написаны автотесты для сервиса Stellar Burgers. 
Локаторы вынесены и разделены по группам. Группы разделены по местам расположения локаторов - страница формы регистрации, главная страница, страницы формы авторизации, страница восстановления доступа, страница с личным кабинетом. 
В фикстуру вынесена подготовка браузера к выполнению тестов.
